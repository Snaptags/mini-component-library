# ProgressBar

Our ProgressBar component accepts a `value` prop, and supports numbers from 0 to 100. There are no animations.

<Story name="Default" args={{ value: 50, size: 'large' }}>
    {ProgressBar.bind({})}
</Story>

It is accessible.

The ProgressBar comes in 3 sizes, note to inner padding of the largest size:

    <DocsLayout title="Large"><ProgressBar value={50} size="large" /></DocsLayout>
</Story>
